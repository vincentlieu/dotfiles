# vincentlieu dotfiles

## What does the script set up?
1. Installs Homebrew & packages with Brewfile
2. Installs oh-my-zsh & zsh plugin and themes
3. Configures terminal - aliases, theme, git, vimrc
4. Configures iTerm2 and hacked nerd fonts
5. Configures nvm with npm packages to be installed alongside
6. Configures zsh fast-syntax-highlighting plugin theme
